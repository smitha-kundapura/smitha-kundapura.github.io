# Smitha Kundapura - Portfolio Website

## üöÄ Quick Start

This portfolio is built with simple HTML, CSS, and JavaScript - no build tools required!

### Files Included:
- `index.html` - Main portfolio page
- `styles.css` - Styling and animations
- `script.js` - Interactive features

### How to Deploy to GitHub Pages:

1. **Upload Files to Your Repository**
   - Go to your repository: `smitha-kundapura.github.io`
   - Click "Add file" ‚Üí "Upload files"
   - Drag and drop all three files (index.html, styles.css, script.js)
   - Click "Commit changes"

2. **Enable GitHub Pages** (if not already enabled)
   - Go to repository Settings
   - Scroll to "Pages" section
   - Under "Source", select "main" branch
   - Click "Save"

3. **View Your Site**
   - Your site will be live at: `https://smitha-kundapura.github.io`
   - It may take 1-2 minutes for changes to appear

## üé® Customization

### Update Your Information:
- Open `index.html` in any text editor
- Modify text content (your name, experience, skills, etc.)
- Update links (LinkedIn, GitHub, etc.)

### Change Colors:
- Open `styles.css`
- Look for `:root` section at the top
- Modify CSS variables (colors start with `--`)

### Add/Remove Sections:
- Each section is wrapped in `<section>` tags
- Copy and paste sections to reorder or duplicate
- Delete sections you don't need

## üìù Adding New Experience

To add a new job:
1. Copy an existing `.timeline-item` block in `index.html`
2. Update the company, title, dates, and achievements
3. Save and push to GitHub

## üîß Troubleshooting

**Site not updating?**
- Clear your browser cache (Ctrl+Shift+R or Cmd+Shift+R)
- Wait 2-3 minutes for GitHub Pages to rebuild
- Check that files are in the root directory (not in a folder)

**Broken styling?**
- Ensure all three files are in the same directory
- Check file names are exactly: `index.html`, `styles.css`, `script.js`
